# Rock-Paper-Scissors Detection

基于计算机视觉的剪刀石头布手势实时识别系统

[English](#english) | [中文](#中文)

## 中文

### 项目描述

本项目实现了一个实时手势识别系统，能够准确识别剪刀、石头、布三种手势。采用深度学习技术，支持实时视频流处理，具有高准确度和低延迟的特点。

### 主要功能

- ✅ 实时手势识别（支持摄像头输入）
- ✅ 支持剪刀、石头、布三种手势检测
- ✅ 高准确率识别模型
- ✅ 低延迟实时处理

### 快速开始

#### 环境要求

- Python 3.8+
- 系统摄像头

#### 安装

```bash
git clone https://github.com/imjouch/rps-detection.git
cd rps-detection
pip install -r requirements.txt
```

#### 使用

```bash
python main.py
```

## English

### Project Description

This project implements a real-time hand gesture recognition system that accurately identifies rock, paper, and scissors gestures. Leveraging deep learning technology, it supports real-time video stream processing with high accuracy and low latency.

### Features

- ✅ Real-time gesture recognition (camera input support)
- ✅ Detection of rock, paper, and scissors gestures
- ✅ High-accuracy recognition model
- ✅ Low-latency real-time processing
